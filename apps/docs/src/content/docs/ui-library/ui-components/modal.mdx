A `Modal` is a dialog that appears on top of the main content. It has a header
with a title, optional subtitle and icon, a body, and an optional footer, and
fades in and out via the Base UI transition lifecycle.

<ModalExample client:visible />

## Usage

```tsx
import { Button, Modal } from '@workflowbuilder/ui';

function Example() {
  const [open, setOpen] = useState(false);
  return (
    <>
      <Button variant="primary" onClick={() => setOpen(true)}>
        Open modal
      </Button>
      <Modal
        open={open}
        onClose={() => setOpen(false)}
        title="Example modal"
        subtitle="Rendered live from @workflowbuilder/ui"
        footer={
          <Button variant="primary" onClick={() => setOpen(false)}>
            Got it
          </Button>
        }
      >
        Modal body content.
      </Modal>
    </>
  );
}
```

## Props

| Prop            | Type                          | Default        | Description                                    |
| --------------- | ----------------------------- | -------------- | ---------------------------------------------- |
| `title`         | `string`                      | -              | Title displayed in the modal header.           |
| `subtitle`      | `string`                      | -              | Optional subtitle displayed below the title.   |
| `children`      | `ReactNode`                   | -              | Content displayed in the modal body.           |
| `footer`        | `ReactNode`                   | -              | Content displayed in the modal footer.         |
| `icon`          | `ReactNode`                   | -              | Optional icon displayed in the header.         |
| `size`          | `'regular' \| 'large'`        | `'regular'`    | Size variant of the modal.                     |
| `footerVariant` | `'integrated' \| 'separated'` | `'integrated'` | Footer styling - flush or divided by a border. |
| `open`          | `boolean`                     | -              | Controls the visibility of the modal.          |
| `onClose`       | `() => void`                  | -              | Called when the modal is closed.               |
| `className`     | `string`                      | -              | Class applied to the modal content body.       |

The component also forwards the remaining native `div` attributes.

## CSS variables

Override these `--ax-*` custom properties to retune the modal.

### Size

| Variable                                     | Description                    |
| -------------------------------------------- | ------------------------------ |
| `--ax-public-modal-width`                    | Modal width (regular).         |
| `--ax-public-modal-large-width`              | Modal width (large).           |
| `--ax-public-modal-padding`                  | Header padding (regular).      |
| `--ax-public-modal-large-padding`            | Header padding (large).        |
| `--ax-public-modal-border-radius`            | Corner radius.                 |
| `--ax-public-modal-content-padding`          | Body padding.                  |
| `--ax-public-modal-footer-padding`           | Footer padding.                |
| `--ax-public-modal-icon-padding`             | Header icon padding (regular). |
| `--ax-public-modal-large-icon-padding`       | Header icon padding (large).   |
| `--ax-public-modal-icon-border-radius`       | Header icon radius (regular).  |
| `--ax-public-modal-icon-border-radius-large` | Header icon radius (large).    |
| `--ax-public-modal-gap`                      | Header gap (regular).          |
| `--ax-public-modal-gap-large`                | Header gap (large).            |
| `--ax-public-modal-content-gap`              | Gap between body items.        |
| `--ax-public-modal-description-size`         | Subtitle font size.            |
| `--ax-public-modal-description-line-height`  | Subtitle line height.          |
| `--ax-public-modal-large-title-size`         | Title font size (large).       |
| `--ax-public-modal-large-title-line-height`  | Title line height (large).     |
| `--ax-public-modal-box-shadow`               | Modal drop shadow.             |
