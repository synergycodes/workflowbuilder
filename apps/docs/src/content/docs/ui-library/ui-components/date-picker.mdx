A `DatePicker` lets users pick a date from a calendar popover. It supports single
dates (the default), date ranges, and multiple-date selection, and renders the
selected value using `date-fns` format tokens.

<DatePickerExample client:visible />

## Usage

```tsx
import { DatePicker } from '@workflowbuilder/ui';

function Example() {
  const [date, setDate] = useState<Date | null>(null);
  return (
    <DatePicker
      value={date ?? undefined}
      placeholder="dd/mm/yyyy"
      valueFormat="dd-MM-yyyy"
      onChange={(next) => setDate((next as Date | null) ?? null)}
    />
  );
}
```

## Props

| Prop              | Type                                                      | Default        | Description                                |
| ----------------- | --------------------------------------------------------- | -------------- | ------------------------------------------ |
| `value`           | `Date \| [Date, Date] \| Date[] \| string`                | -              | Controlled selected value.                 |
| `defaultValue`    | `Date \| [Date, Date] \| Date[] \| string`                | -              | Initial value when uncontrolled.           |
| `onChange`        | `(value: Date \| [Date, Date] \| Date[] \| null) => void` | -              | Called when the selection changes.         |
| `type`            | `'default' \| 'range' \| 'multiple'`                      | `'default'`    | Single date, a range, or multiple dates.   |
| `valueFormat`     | `string`                                                  | `'DD/MM/YYYY'` | `date-fns` token used to render the value. |
| `placeholder`     | `string`                                                  | `'dd/mm/yyyy'` | Text shown when no date is selected.       |
| `inputSize`       | `'small' \| 'medium' \| 'large'`                          | `'medium'`     | Size of the trigger input.                 |
| `minDate`         | `Date`                                                    | -              | Earliest selectable date (inclusive).      |
| `maxDate`         | `Date`                                                    | -              | Latest selectable date (inclusive).        |
| `error`           | `boolean`                                                 | `false`        | Renders the trigger in an error state.     |
| `disabled`        | `boolean`                                                 | -              | Disables the trigger and prevents opening. |
| `readOnly`        | `boolean`                                                 | -              | Shows the value but prevents opening.      |
| `id`              | `string`                                                  | -              | `id` applied to the trigger button.        |
| `className`       | `string`                                                  | -              | Class applied to the trigger button.       |
| `aria-label`      | `string`                                                  | -              | Accessible name for the trigger button.    |
| `aria-labelledby` | `string`                                                  | -              | `aria-labelledby` for the trigger button.  |

## CSS variables

Override these `--ax-*` custom properties to retune the date picker.

### Size

| Variable                                 | Description                     |
| ---------------------------------------- | ------------------------------- |
| `--ax-public-input-padding-large`        | Trigger padding (large).        |
| `--ax-public-input-padding-medium`       | Trigger padding (medium).       |
| `--ax-public-input-padding-small`        | Trigger padding (small).        |
| `--ax-public-input-border-radius-large`  | Trigger corner radius (large).  |
| `--ax-public-input-border-radius-medium` | Trigger corner radius (medium). |
| `--ax-public-input-border-radius-small`  | Trigger corner radius (small).  |
| `--ax-public-input-root-border-size`     | Border width of the trigger.    |
