`Button` is a flexible, type-safe component that automatically selects the
correct type (label, icon, or icon + label) based on the structure of its
`children`. A single string renders a label button, a single React element
renders an icon button, and a string combined with icons renders an icon-label
button.

<ButtonExample client:visible />

## Usage

```tsx
import { Button } from '@workflowbuilder/ui';

function Example() {
  return <Button variant="primary">Submit</Button>;
}
```

## Props

These props apply to the label button shown above. Icon and icon-label variants
accept the same base props plus their own icon props.

| Prop          | Type                                                                                             | Default     | Description                  |
| ------------- | ------------------------------------------------------------------------------------------------ | ----------- | ---------------------------- |
| `children`    | `string`                                                                                         | -           | Button label.                |
| `variant`     | `'primary' \| 'secondary' \| 'gray' \| 'error' \| 'warning' \| 'success' \| 'ghost-destructive'` | `'primary'` | Visual style of the button.  |
| `size`        | `'extra-small' \| 'small' \| 'medium' \| 'large' \| 'extra-large'`                               | `'medium'`  | Size of the button.          |
| `isLoading`   | `boolean`                                                                                        | `false`     | Shows a loading indicator.   |
| `tooltip`     | `string`                                                                                         | -           | Tooltip text.                |
| `tooltipType` | `TooltipVariant`                                                                                 | -           | Tooltip style variant.       |
| `disabled`    | `boolean`                                                                                        | `false`     | Disables interaction.        |
| `className`   | `string`                                                                                         | -           | Class applied to the button. |

The component also forwards the remaining native `button` element attributes.
